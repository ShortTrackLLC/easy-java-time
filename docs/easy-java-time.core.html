chrono-unit

(chrono-unit unit)

Returns java.time.temporal.ChronoUnit/UNIT for shorthand unit. All units except :millennia and :forever can be provided in singular or plural form - this is just for readability and has no effect on the value returned.

Unit
:day, :days
:decade, :decades
:hour, :hours
:micro, :micros
:milli, :millis
:minute, :minutes
:month, :months
:second, :seconds
:week, :weeks
:year, :years

Examples:

(require '[easy-java-time.core :as et])

inst->local-date-time

(inst->local-date-time instant & [zone-offset])

Returns a java.time.LocalDateTime for instant. zone-offset defaults to java.time.ZoneOffset/UTC.

Examples:

(require '[easy-java-time.core :as et])

(et/inst->local-date-time (et/now))
;= #object[java.time.LocalDateTime 0x933275d "2024-11-29T03:45:02.302190"]

now

(now)

Returns a java.time.Instant at present in UTC.

Examples:

(require '[easy-java-time.core :as et])

(et/now)
;= #object[java.time.Instant 0x776e1ae5 "2024-11-29T03:32:48.798592Z"]

str->inst

(str->inst s)

Returns a java.time.Instant after parsing string s.

Examples:

(require '[easy-java-time.core :as et])

(et/str->inst "2024-11-29T03:32:48.798592Z")
;= #object[java.time.Instant 0x41694e7a "2024-11-29T03:32:48.798592Z"]
